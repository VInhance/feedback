# Feedback Hub

Welcome to the **VInhance** feedback hub. 

This repository is a dedicated space for users to report bugs and suggest features for VInhance. 

---

## 🛠️ How to Contribute

If you've encountered a bug or have an idea to make VInhance better, please follow these steps:

### 1. Check Existing Issues
Before opening a new issue, please use the search bar in the [Issues tab](https://github.com/VInhance/feedback/issues) to see if someone else has already reported the same thing.

### 2. Opening an Issue
If your issue hasn't been reported:
- **Bug Reports:** Provide a clear description of the problem, steps to reproduce it, and screenshots if possible. Mention your Campus (Vellore, Chennai, Bhopal, or AP).
- **Feature Requests:** Describe the feature you'd like to see and how it would improve the VTOP experience.

[🚀 Open a New Issue](https://github.com/VInhance/feedback/issues/new/choose)
